    Raise a synchronous event for a given state machine.
    First checks if this state machine expects any events of this type for the state machine's CurrentState. These would follow an attribute of type OnEventDoActionAttribute.
    If the state machine's CurrentState expects a transition based on the event, then this will trigger the OnExitStateAttribute. of the CurrentState, change to the next state defined in the OnEnterStateAttribute. of the next state.
